# Security Policy

## Reporting a Vulnerability

Please report security vulnerabilities through GitHub's private vulnerability reporting feature:

1. Go to the [Security tab](https://github.com/groundnuty/claude-plan-composer/security)
2. Click "Report a vulnerability"
3. Provide a description of the issue, steps to reproduce, and any relevant details

**Do not file public issues for security vulnerabilities.**

## Scope

- Claude Plan Composer CLI and SDK code
- Dependencies bundled with the package

Out of scope:
- The Claude API itself (report to [Anthropic](https://www.anthropic.com/security))
- Third-party dependencies (report to the upstream project)

## Response

We aim to acknowledge reports within 48 hours and provide a fix timeline within 7 days.
