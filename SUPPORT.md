# Support

## Bugs and Feature Requests

File an [issue](https://github.com/groundnuty/claude-plan-composer/issues) using the appropriate template:

- **Bug report** — something isn't working as expected
- **Feature request** — suggest a new capability
- **Plan quality issue** — the merged plan had an unexpected problem

## Questions and Discussion

Use [GitHub Discussions](https://github.com/groundnuty/claude-plan-composer/discussions):

- **Q&A** — how do I configure X?
- **Ideas** — what if CPC could do Y?
- **Show and Tell** — share how you're using CPC
